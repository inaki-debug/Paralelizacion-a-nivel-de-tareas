# Paralelizaci-n-a-nivel-de-tareas
Esta actividad consistió en la creación de 4 funciones las cuales modificaban la estructura de una imagen .bmp. La finalidad es poder paralelizar el proceso principal de cada función y aplicar estas 4 funciones a diferentes imágenes al mismo tiempo. También el objetivo fue paralelizar a nivel de tareas cada llamada a la función que engloba las 4 funciones. 
Para más información, consultar la wiki de este repositorio.
